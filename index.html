Autonomous Relay Matrix (AI Steering)

The streaming proxy automatically parses WebSocket HTTP Upgrade headers and intercepts the SNI payload. If a user requests AI-powered endpoints (OpenAI, Anthropic, Claude) and your origin edge relies on Cloudflare's general ingress mapping, this system autonomously intercepts the connect() wrapper and steers the connection towards physical US-based endpoints flagged in the D1 store below.

Currently enforced exclusively at the Edge Network via `cloudflare:sockets` byte interceptor engine.
